# Procura CEP

Procura CEP é um projeto em node que faz a busca de CEPs.

## Instalação

Use o [npm](https://www.npmjs.com/) para instalar as bibliotecas necessárias ao projeto.

```bash
npm i
```

## Rodando o projeto

Use o comando:

```bash
npm run start
```
